MolluscaGenes — preliminary release (v0.1)

A mollusc transcriptome and proteome resource, paired with the TIAMMAt mollusc-revised Pfam HMMs (190 domains across 50 biological categories), plus command-line wrappers for BLAST / DIAMOND / hmmsearch search, species extraction, and CrusTome-style iterative-BLAST phylogenetic placement.

v0.1 is a preliminary release of the database used in the accompanying biorxiv preprint. A full rebuild (v1.0) is in progress on Unity HPC and will supersede v0.1 under the same Zenodo concept DOI.

What's in the release

  • ~333 mollusc species across all major classes (Gastropoda, Bivalvia, Cephalopoda, Scaphopoda, Solenogastres, Caudofoveata, Polyplacophora, Monoplacophora) — 212 with sequences in this release, 98 listed transparently as planned-but-pending.
  • BLAST databases (protein + nucleotide) and a DIAMOND protein database built from the same source.
  • TIAMMAt mollusc-revised HMMs: 190 Pfam profiles re-trained on mollusc-derived sequences. Tagged with biological categories (GPCR, VGIC, LGIC, etc.).
  • Per-species and per-HMM metadata tables with NCBI taxonomy and WoRMS cross-links.
  • Wrapper scripts: mg_blast, mg_diamond, mg_hmmsearch, mg_characterize, mg_place (CrusTome-style phylogeny), mg_extract (per-species or per-taxon FASTA extraction).

Download

The database files (~13 GB compressed) are hosted on Zenodo. The concept DOI always resolves to the latest version; the v0.1 DOI permanently pins this release.

Recommended workflow (clone repo, then fetch with the wrapper):

git clone https://github.com/invertome/molluscagenes
cd molluscagenes
conda env create -f environment.yml
conda activate molluscagenes
bash wrappers/mg_fetch.sh /path/to/storage   # downloads, verifies, writes config.sh
source config.sh
bash wrappers/mg_blast.sh -q my_query.fa -o blast_out -d aa
