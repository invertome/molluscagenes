MolluscaGenes — a transcriptomic database for the Mollusca

A taxonomically comprehensive resource consolidating de-novo and previously published transcriptomes for ~300 mollusc species across all eight classes, paired with the TIAMMAt mollusc-revised Pfam HMMs and command-line wrappers for BLAST / DIAMOND / hmmsearch search, species extraction, and iterative-BLAST phylogenetic placement.

v0.1 — preliminary release accompanying the biorxiv preprint. A full HPC rebuild (v1.0) is in progress and will supersede v0.1 under the same Zenodo concept DOI. Both code (GPL-3.0) and data (CC-BY-4.0) are open.

Why MolluscaGenes

Mollusca, the second-largest animal phylum (>70,000 described species across eight classes), has historically been under-served by genomic resources. Mollusc transcriptomes are scattered across NCBI BioProjects, MolluscDB, MateDB, and individual lab repositories. Detection of divergent protein homologs across the phylum is further constrained by the bias of public HMM resources (e.g. Pfam) toward vertebrate and ecdysozoan sequences, which can miss legitimate molluscan family members because of lineage-specific substitution patterns.

MolluscaGenes addresses both problems: it consolidates ~300 species worth of transcriptomic data into a single searchable resource, and ships mollusc-optimized HMMs — Pfam profiles iteratively re-trained against molluscan sequence diversity for substantially higher sensitivity on lophotrochozoan homologs.

What's in the release

  • ~300 species spanning all eight molluscan classes (Gastropoda, Bivalvia, Cephalopoda, Polyplacophora, Scaphopoda, Solenogastres, Caudofoveata, Monoplacophora).
  • ~17 million transcript sequences (~16.8 Gb of nucleotide data).
  • ~17 million predicted protein sequences (~3.3 Gb of amino-acid data).
  • BLAST databases (protein + nucleotide) and a DIAMOND protein database built from the same source.
  • 190 mollusc-revised HMMs across 50 biological categories (innate immunity, developmental signalling, Ca²⁺ signalling, voltage- and ligand-gated ion channels, GPCRs, shell biomineralization, epigenetic regulation, chemoreception, circadian rhythms, neuropeptide processing, toxin biology, synaptic function, and more) — see the HMM browser.
  • Per-species and per-HMM metadata tables with NCBI taxonomy and WoRMS cross-links — see the species browser.
  • Wrapper scripts: mg_blast, mg_diamond, mg_hmmsearch, mg_characterize, mg_place (iterative-BLAST phylogenetic placement), mg_extract (per-species or per-taxon FASTA extraction).

How v0.1 was built

Raw reads were sourced from the NCBI Sequence Read Archive (paired-end Illumina, ≥20M read-pairs/sample, prioritizing tissue diversity and under-represented classes), supplemented with pre-assembled transcriptomes from MateDB and MolluscDB. The pipeline is the nf-core/denovotranscript multi-assembler workflow:

  1. QC: fastp (Q20 sliding window, min length 50, low-complexity filter).
  2. Assembly: Trinity + rnaSPAdes (k=25,49,73), merged.
  3. Redundancy reduction: EvidentialGene (98% clustering, retains main + alternate transcripts).
  4. Quality gate: BUSCO (Metazoa odb10) — assemblies below 30% completeness excluded.
  5. Annotation: DIAMOND blastp vs. NCBI RefSeq + InterProScan (Pfam, SMART, CDD, SUPERFAMILY, Gene3D) + eggNOG-mapper.
  6. HMM optimisation: TIAMMAt iterative refinement (initial Pfam → hmmsearch → MAFFT E-INS-i → hmmbuild → repeat 3–5×) validated against six held-out proteomes (Aplysia californica, Crassostrea gigas, Lingula anatina, Lottia gigantea, Octopus bimaculoides, Pomacea canaliculata).

Recommended workflow — clone the repo, then fetch + verify with the wrapper:

git clone https://github.com/invertome/molluscagenes
cd molluscagenes
conda env create -f environment.yml
conda activate molluscagenes
bash wrappers/mg_fetch.sh /path/to/storage   # downloads, verifies, writes config.sh
source config.sh
bash wrappers/mg_blast.sh -q my_query.fa -o blast_out -d aa
